Tab Navigation 컴포넌트 예시

상단의 탭을 클릭해보세요! (현재는 알림창으로 표시됩니다)

현재 활성 탭: option

사용된 코드:

const tabNav = new TabNavigation({
    container: document.getElementById('navigation-container'),
    activeTab: 'option',
    onTabChange: (tabKey) => {
        console.log('탭 변경:', tabKey);
        document.getElementById('current-tab').textContent = tabKey;
        alert(`${tabKey} 탭으로 이동합니다!`);
    }
});
