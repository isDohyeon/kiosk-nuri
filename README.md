# 🏪 키오스크 누리 (Kiosk Nuri) - 웹 키오스크 시스템

**2025 CPD(Creative Product Design) - Team ReVerse**

테크노스트레스를 겪는 디지털 소외계층들도 쉽게 사용할 수 있는 키오스크를 디자인

## 🎯 화면 플로우

```
메인 화면 (index.html)
    ↓
메뉴 선택 (pages/menu/)
    ↓
옵션 선택 (pages/option/)
    ↓
할인 선택 (pages/discount/)
    ↓
적립 선택 (pages/point/)
    ↓
결제 화면 (pages/payment/)
    ↓
완료 화면 (pages/complete/)
```
